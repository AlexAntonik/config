### **NixOS Configuration**

#### Requirements

- NixOS
- EFI partition ≥ 512MB

#### Installation

Install git,curl:
```bash
nix-shell -p git curl
```

Run install script:
```bash
sh <(curl -L https://raw.githubusercontent.com/AlexAntonik/config/refs/heads/master/install.sh)
```
